QH Stock Intel Notification Pipeline

Updated: December 2025

This project delivers automated stock-related notification emails to subscribed users. The system analyzes selected publicly available data sources and sends periodic alerts or summaries.

How It Works

  • Users provide an email address to receive notification messages.
  • No login or authentication is required.
  • All outbound emails are sent from a Google account controlled solely by the developer.
  • The Gmail API is used only to send messages. Users do not grant Gmail access of any kind.
