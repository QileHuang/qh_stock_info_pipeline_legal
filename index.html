StockIntel Automated Email Pipeline

StockIntel is a personal project that automates daily stock sentiment summaries and dispatches them to the user's Gmail inbox. It analyzes online discussions, aggregates signals, and delivers a concise overview.

Why This App Uses Google OAuth

The application uses Gmail API access only to send automated emails on behalf of the developer. It does not read, store, or access the existing email content.

Data Handling

The app does not store personal data on any external server. All processing takes place locally on the developer-controlled environment. No email content, tokens, or metadata is shared or sold.
